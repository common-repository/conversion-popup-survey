Direct traffic on your blog by using simple popup surveys.

== Description ==

With conversion popup survey you can direct traffic to certain pages (URLs) by using a simple popup surveys.

You can fully customize the popup and its contents look and feel.

It also comes with a delay timer and number of times a certain popup is shown.

You can have unlimited number of popup surveys on your blogs pages and posts.

== Installation ==

1. Download & extract plugin using its default folder
1. Upload plugin folder to the "/wp-content/plugins/" folder. If you're uploading it make sure to upload the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`
1. Activate the plugin through the "Plugins" menu in WordPress
1. That's it. A new option will be shown in the menu of the WP Admin area
